A special interactive web app to celebrate your love!

## Customize Your Journey

Edit `app/page.js` to personalize:
- Questions in the `stages` array
- Answer options
- Correct answer indices (0-3)
- Sweet messages after each correct answer
- Final love message

## Deploy to Vercel

1. Push this code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Click "Deploy"

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000
